# このスクリプトについて
Githubのリポジトリからファイル・フォルダーをコピーするNode.jsのスクリプトです。`.git`フォルダーは作成されないため、Gitコマンドは実行できません。(これを使う時点でGitは使えない想定だから当然)
# 使い道
ホスティングサービスなどでGithubからファイルたちを持って来れない時に使えます。それだけです。
# 使用方法
ソースを丸々ダウンロードし、`.env`ファイルを作成してください。  
設定は以下です。
```
// プライベート(非公開の)リポジトリの場合、トークンが必要
token=トークン
// 以下は必須
owner=ユーザー名
repo=リポジトリ名
// 任意
source=出力先のフォルダーのパス
```
<span style="font-size: 12px;"><a href="https://qiita.com/YuukiYoshida/items/2e6b250d44bf1e0f5a0b">トークンの発行の方法</a> ※他の方の記事です</span>
  
ターミナルで`npm run fetch`を実行すると、`.env`で`source`を指定した場合にはそのフォルダー、指定しなかった場合には`REPO`フォルダーに出力されます。
